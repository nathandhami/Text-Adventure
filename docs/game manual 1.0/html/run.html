Launching The Game¶

How to Launch¶

In projects root folder:

To start the server, run: ./server-start.sh

To start the game, run: ./play.sh

Example¶

Run these commands:

cd /textadventure ./server-start.sh

Then open a new terminal:

cd /textadventure ./play.sh
